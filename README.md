# 简介
课程由 Heather Miller 主讲，4周课程。 通过本课程，你将学习：
- 利用 Spark，将数据并行范式扩展至分布式
- Spark 的编程模型
- Spark中的分布式计算、集群拓扑
- 如何提高性能：数据本地化， 在 Spark 中如何避免重新计算及 shuffle
- DataFrame 与 DataSet 的关系型API操作
